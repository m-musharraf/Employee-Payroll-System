## Overview

A desktop-based Employee Payroll System developed using C#, .NET, and SQL Server. This project was developed as an academic project for the Higher National Diploma (HND) in Computing.


---

## Technologies Used

- C#
- .NET Framework
- SQL Server

---

## Features

- Employee Registration (CRUD)
- Employee Leave and Payroll Management
- Search Functionality
- Reports
- User Login
- Dashboard
